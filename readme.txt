Enables csv file upload and CRUD for database named "artisans"

== Description ==
This plugin creates predefined database tables to enable the 
display of an excel (format csv sheet) which is a directory of
the artisans registered with the CMA 46. It is possible to search the database of artisans by their location and speciality.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Tools -> Annuaire_artisans -> Uploading Files screen to use the plugin.
