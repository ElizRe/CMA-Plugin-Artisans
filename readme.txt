Enables the transfer of CSV files and CRUD for the database named  "Artisans "

Active le transfert de fichiers CSV et CRUD pour la base de données nommée "Artisans"

== Description ==
This plugin creates predefined database tables to enable the 
display of an excel (format csv, tab separated) which is a directory of the artisans registered with the CMA 46. It is possible to search the database of artisans by their location and speciality.
Ce plugin crée des tables de base de données prédéfinies pour permettre l'affichage d'une Excel (formaté CSV onglet séparé) qui est un répertoire de les artisans inscrits à la CMA 46. Il est possible de rechercher dans la base de données des artisans par leur emplacement et spécialité

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Tools -> Annuaire_artisans -> Uploading Files screen to use the plugin.

1.Installez le plugin à travers l'écran WordPress plugins directement.
2. activer le plugin à travers l'écran'plugins'dans WordPress.
3. Utilisez l'écran Tools-> Annuaire_artisans-> uploading files pour utiliser le plugin.
